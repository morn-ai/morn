# Morn Agent

[English](README.md) | 简体中文

## 快速开始

### 前置要求

- Python 3.9 或更高版本
- [uv](https://github.com/astral-sh/uv)（推荐）或 pip

### 安装

#### 使用 uv（推荐）

```bash
# 克隆仓库
git clone https://github.com/morn-ai/morn-agent.git
cd morn-agent

# 安装依赖
uv sync

# 运行开发服务器
uv run python app/app.py
```

#### 使用 pip

```bash
# 克隆仓库
git clone https://github.com/morn-ai/morn-agent.git
cd morn-agent

# 安装依赖
pip install -e .

# 运行开发服务器
python app/app.py
```
